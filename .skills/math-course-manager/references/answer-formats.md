# 答案格式参考

## 课后小测格式

```markdown
> 题目原图见下方，答案折叠隐藏。做错的题到 [[错题 第N讲 {主题}]] 中记录。

## 一、填空题
![[第N讲_课后小测_题目_p1.png]]

### 第1题
> [!question] 简短标题
> 完整题目文字（含具体数字）

- [x] 已掌握  （或 - [ ] 未掌握 → [[错题链接]]）

> [!answer]- 点击查看答案
> **答案**：xxx
> 解析步骤
> **知识点**：[[第N讲 {主题}#知识点名]]
```

## 加油站格式

```markdown
## 题目（含手写解析）
![[第N讲_加油站_解析_p1.png]]
...

## 答案汇总
> [!answer]- 举一反三
> | 题号 | 答案 |
> |------|------|
> | 1 | xxx |

> [!answer]- 基础过关
...

## 掌握情况
**举一反三**
- [ ] 第1题
...
```

## 错题本格式

```markdown
### 错题 #N — 简短描述
- **来源**：课后小测/加油站 第X题
- **日期**：YYYY-MM-DD
- **错因**：计算错误 / 方法不对 / 审题不清 / 知识点遗忘
- **题目**：完整题目
- **正确思路**：分步骤写清楚
- **关键**：一句话总结核心方法
- **关联知识点**：[[第N讲 {主题}#知识点名]]

## 复习记录
| 日期 | 出处 | 结果 | 备注 |
|------|------|------|------|
| YYYY-MM-DD | 变式练习 第N讲 第X题 | ✅ 掌握 | 首次变式练习即做对 |
```

## 变式练习格式

```markdown
### 第X题（针对的薄弱点）
> [!question]
> 完整题目

**我的答案**：

> [!answer]- 点击查看答案
> **答案**：xxx
> 解析步骤
> **知识点**：[[第N讲 {主题}#知识点名]]

- [ ] 正确
- [ ] 错误 → 记录到 [[错题 第N讲 {主题}]]

### 第X题（YYYY-MM-DD）
> [!question]
> 题目内容

**小朋友的答案**：xxx
**结果**：✅ 正确 / ❌ 错误
**引导过程**：（如有）
> 提示1：...
> 小朋友第2次回答：...
> 提示2：...

> [!answer]- 点击查看答案
> 完整解析

- [x] 正确 / - [ ] 错误 → 记录到 [[错题 第N讲 {主题}]]
```
