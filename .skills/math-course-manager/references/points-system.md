# 答题积分体系

虫虫有完整的积分体系，每次答题互动后必须同步更新。文件位于 `C:\Users\hp\xiaochong-math\积分体系\`。

## 积分文件说明

| 文件 | 用途 |
|------|------|
| `积分账本.md` | 记录所有积分增减明细 |
| `成就墙.md` | 徽章获得记录 |
| `连胜记录.md` | 每日一题连续天数追踪 |
| `周排名.md` | 每周积分排行 |

## 积分规则

### 基础积分

| 来源 | 规则 | 积分 |
|------|------|------|
| 课后小测 | 6道题总100分，得多少分加多少积分 | 0~100分 |
| 每日一题 | 答对一道题 | +10积分 |

### 成就奖励

| 徽章 | 名称 | 条件 | 奖励积分 |
|------|------|------|---------|
| 🌟 | 首次满分 | 第一次小测100分 | +50分 |
| 🔥 | 3日连胜 | 连续3天每日一题 | +30分 |
| 🔥 | 7日连胜 | 连续7天每日一题 | +50分 |
| 🔥 | 月度达人 | 连续30天每日一题 | +200分 |
| 💪 | 攻破弱点 | 某错题连续2次变式做对 | +30分 |
| 📚 | 好学新星 | 单周完成5次每日一题 | +40分 |

## 互动出题后必须执行

1. **更新积分账本 → 每日一题表格**：记录日期、题目、情况、积分
2. **更新连胜记录**：今日完成则连续天数+1，否则归零
3. **检查成就墙**：是否达成新徽章条件，若有则更新「已获徽章」表格
4. **更新周排名**：每周一结算
