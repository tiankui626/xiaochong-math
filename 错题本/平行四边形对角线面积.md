# 错题记录

## 题目特征

平行四边形被两条对角线分割成4个小平行四边形，已知其中3个的面积，求第4个。

```
   ┌───────┬───────┐
   │  S1   │  S2   │
   │       │       │
   ├───────┼───────┤
   │  S3   │  S4   │
   │       │       │
   └───────┴───────┘
```

## 正确关系

**S1 × S4 = S2 × S3**（对角线配对相乘）

注意：配对是 **S1配S4，S2配S3**，不是 S1配S3。

## 常见错误

### 错因1：记成相加
- 错误：S1 + S3 = S2 + S4
- 正确：S1 × S4 = S2 × S3

### 错因2：配对错误
- 错误：S1×S3 和 S2×S4 配对
- 正确：S1×S4 和 S2×S3 配对

## 替代解法（不记公式也能做）

**核心思路：整体思想 + 方程思想**

把平行四边形想象成一个矩形被分割成4个小格：

```
   ┌─────────┬─────────┐
   │   a     │   b     │
   │   x     │   x     │
   ├─────────┼─────────┤
   │   c     │   d     │
   │   y     │   y     │
   └─────────┴─────────┘
```

每个小格的面积 = 长 × 宽，所以：
- S1 = a × x
- S2 = b × x
- S3 = c × y
- S4 = d × y

关键发现：**a/b = S1/S2 = S3/S4**（同一行高度相同）

因此：S1/S2 = S3/S4 → S1 × S4 = S2 × S3

**就算记不住公式，用这个方法也能推出来！**

## 思维卡点

- [ ] 没有想到配对相乘的关系
- [ ] 记错了公式（相加而非相乘）
- [ ] 配对记错（S1配S3 而非 S1配S4）
- [ ] 没有想到用整体思想：把每个小格看成"长×宽"，利用比例关系推导

## 转化思想关联

这道题是**转化思想**的经典案例：
- "把图形分割转化为代数比例"
- "把未知转化为已知，用方程思想求解"
- "不背公式，用构造法推导"
