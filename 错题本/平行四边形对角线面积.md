# 错题记录

## 题目特征

平行四边形被两条对角线分割成4个小平行四边形，已知其中3个的面积，求第4个。

```
   ┌───────┬───────┐
   │  S1   │  S2   │
   │       │       │
   ├───────┼───────┤
   │  S3   │  S4   │
   │       │       │
   └───────┴───────┘
```

## 正确关系

**同一底的两个三角形面积比 = 另一底的两个三角形面积比**

即：同一行（或同一列）的两个小格，面积比等于另一行（或另一列）的两个小格面积比。

推导：每个小格面积 = 底 × 高，所以 S1/S2 = S3/S4（同一高的情况下）

配对关系取决于图形结构，要根据具体题目判断哪两个是对角线配对。

**核心：不是背公式，是用"长×宽"构造比例关系推导。**

## 常见错误

### 常见错误

- 写成加法关系（S1+S3 = S2+S4）——应该是比例关系
- 乱配对，不看图就套公式

## 替代解法（不记公式也能做）

**核心思路：整体思想 + 方程思想**

把平行四边形想象成一个矩形被分割成4个小格，每个小格的面积 = 长 × 宽。

关键发现：同一行（或同一列）的两个小格，底相同所以面积比等于高的比。

因此已知3个面积，可以根据比例关系推导出第4个。

**就算记不住公式，用这个方法也能推出来！**

## 思维卡点

- [ ] 没有想到配对相乘的关系
- [ ] 记错了公式（相加而非相乘）
- [ ] 配对记错（S1配S3 而非 S1配S4）
- [ ] 没有想到用整体思想：把每个小格看成"长×宽"，利用比例关系推导

这道题是**转化思想**的经典案例：
- "把图形分割转化为代数比例"
- "把未知转化为已知，用方程思想求解"
- "不背公式，用构造法推导"
