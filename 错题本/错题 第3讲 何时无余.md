### 错题 #1 — 33的整除特征
- **来源**：课后小测 第2题
- **日期**：2026-04-13
- **错因**：知识点遗忘
- **题目**：六位数 $\overline{32a736}$ 能被33整除，求 $a$
- **正确思路**：
  1. 33的整除特征和99类似：**两位一分段求和**
  2. $32 + \overline{a7} + 36 = 68 + \overline{a7}$
  3. 这个和要能被33整除
  4. 当 $a=9$ 时，$68+97=165$，$165 \div 33 = 5$ ✓
- **关键**：33/99的整除特征不是看末位或数字和，而是**两位一分段求和**
- **关联知识点**：[[第3讲 何时无余#1. 末位判定法]]

### 错题 #2 — 数字和凑9的倍数
- **来源**：课后小测 第4题
- **日期**：2026-04-13
- **错因**：计算错误
- **题目**：在2017右边连续添数码2，第一个9的倍数是？
- **正确思路**：
  1. 原数字和：$2+0+1+7=10$
  2. 每添一个2，数字和增加2
  3. 需要数字和是9的倍数：$10+2k$ 是9的倍数
  4. $k=4$ 时，$10+8=18$ ✓
  5. 答案：20172222
- **关键**：添加数码只影响数字和，不需要算整个大数
- **关联知识点**：[[第3讲 何时无余#2. 数字和判定法]]

### 错题 #3 — 拼接数的整除
- **来源**：课后小测 第6题
- **日期**：2026-04-13
- **错因**：方法不对
- **题目**：1234567891011…写到哪个自然数时，第一次同时被8和9整除？
- **正确思路**：
  1. **被9整除**：各自然数的数字总和是9的倍数 → 等价于各自然数之和是9的倍数
     - 用等差数列求和：$1+2+\cdots+n = n(n+1)/2$
     - $n$ 或 $n+1$ 是9的倍数
  2. **被8整除**：末三位构成的数是8的倍数
  3. 两个条件同时满足：逐一检验 $n=9,17,18,26,27,35,36\cdots$
  4. $n=36$ 时，末三位为536，$536 \div 8 = 67$ ✓
- **关键**：被9整除用数字和（转化为自然数之和），被8整除只看末三位，两个条件独立判断
- **关联知识点**：[[第3讲 何时无余#5. 同时整除多个数]]、[[第3讲 何时无余#6. 大数的余数]]
