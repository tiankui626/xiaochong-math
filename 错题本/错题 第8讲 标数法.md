### 错题 #1 — 跳棋三角形网格
- **来源**：课后小测 第5题
- **日期**：2026-04-29
- **错因**：三角形网格中"走两步"的来源处理不完整
- **题目**：跳棋游戏，每次在向右或向上走一步或两步（两步只能走直线），从A到B有多少种走法？
- **问题分析**：普通标数法只考虑相邻点作为来源，但本题允许走两步（跳一格），所以每个点的来源不仅是相邻的左边/下边点，还包括隔一格的左边/下边点。三角形网格结构复杂，容易漏掉某些来源方向。
- **正确思路**：
  1. 方向：向右、向上（三角形网格中是向右上和向左上）
  2. 每个点的来源：相邻的1步来源 + 隔一格的2步来源（只能直线走2步）
  3. 从A开始逐层标数，每个点 = 所有能一步或两步到达它的点之和
  4. 最终B = 1013
- **关键**：遇到"可走1步或2步"时，来源要扩大范围，不能只看相邻点

### 错题 #2 — 足球比赛进球顺序
- **来源**：课后小测 第6题
- **日期**：2026-04-29
- **错因**：不会将实际问题转化为标数法模型
- **题目**：甲乙两队比分6:2，甲先进1球且乙从未领先，进球顺序有多少种？
- **问题分析**：这道题表面上看不出是标数法，需要"构造模型"——把进球过程转化为网格路径问题。这种从实际场景到数学模型的转化是难点。
- **正确思路**：
  1. 构造阶梯：横一格 = 甲进1球，竖一格 = 乙进1球
  2. 甲进6球乙进2球 → 从起点走到(6,2)的路径
  3. 约束"甲先进1球且乙不领先" → 路径不能越过某条边界线
  4. 甲先进1球 → 第一步固定向右，剩下从(1,0)走到(6,2)
  5. 乙不领先 → 任何时刻甲的进球数 ≥ 乙的进球数 → 路径不能碰到对角线上方
  6. 用标数法在约束下标数 → 20种
- **关键**：看到"顺序""不超过"这类关键词 → 想到构造网格 + 标数法。横竖分别代表两类事件，约束条件变成路径限制
- **关联知识点**：[[第8讲 标数法#2. 基本网格最短路径]]（构造模型）
