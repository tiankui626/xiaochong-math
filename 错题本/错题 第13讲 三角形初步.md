## 课后小测错题

### 错题 #1 — 三边关系求种数
- **来源**：课后小测 第2题
- **日期**：2026-06-15
- **错因**：计算错误
- **题目**：三角形两边为7和12，第三边为整数，共有多少种不同的三角形？
- **正确思路**：
  1. 三边关系：$|7-12| < a < 7+12$，即 $5 < a < 19$
  2. $a$ 为整数：$6, 7, 8, \ldots, 18$
  3. 共 $18 - 6 + 1 = 13$ 种
- **关键**：两边之差 < 第三边 < 两边之和，严格不等号取不到端点
- **关联知识点**：[[第13讲 三角形初步#3. 三角形三边关系]]

### 错题 #2 — 周长偶数求第三边
- **来源**：课后小测 第3题
- **日期**：2026-06-15
- **错因**：分类遗漏（只写了2025，漏掉2027）
- **题目**：三角形周长为偶数，两边为3和2026，求第三边。
- **正确思路**：
  1. 三边关系：$2023 < x < 2029$
  2. 周长 $= 2029 + x$ 为偶数 → $x$ 为奇数
  3. 范围内的奇数：$x = 2025$ 或 $x = 2027$（两个都要写）
- **关键**：先列范围再加约束筛选，满足条件的可能不止一个，别漏写
- **关联知识点**：[[第13讲 三角形初步#3. 三角形三边关系]]
