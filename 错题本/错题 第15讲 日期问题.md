### 错题 #1 — 跨年天数计算
- **来源**：课后小测 第5题
- **日期**：2026-06-24
- **我的错误答案**：星期一
- **正确答案**：星期日
- **错因分析**：闰年计算或总天数算错。1997→2007共10年，其中2000、2004是闰年（多2天），总天数$365\times10+2=3652$，$3652\div7=521$余$5$，星期二往后5天=星期日。
- **正确思路**：
  1. 确定跨了几个闰年：2000、2004（共2个）
  2. 总天数：$365\times10+2=3652$
  3. $3652\div7=521\cdots5$
  4. 星期二 + 5天 = 星期日
- **关键**：跨年问题别忘数闰年！而且余数要从起始星期往后数
- **关联知识点**：[[第15讲 日期问题#跨年问题]]

### 错题 #2 — 周期相遇求星期
- **来源**：课后小测 第6题
- **日期**：2026-06-24
- **我的错误答案**：星期六（算$3\times5\times9=135$天，$135\div7=19$余2）
- **正确答案**：星期二
- **错因分析**：两个错误——①"每隔N天"=每(N+1)天去一次，不是每N天；②即使用135也算错了余数
- **正确思路**：
  1. "隔3天去一次"=每**4**天去一次；"隔5天"=每**6**天；"隔9天"=每**10**天
  2. 求4、6、10的最小公倍数 = **60**天
  3. $60\div7=8$余$4$
  4. 星期五 + 4天 = **星期二**
- **关键**："每隔N天"=周期是(N+1)天！这是经典陷阱
- **关联知识点**：[[第15讲 日期问题#周期性]]

## 易错题型汇总

| 题型 | 易错点 | 对策 |
|------|--------|------|
| "第几天"问题 | 混淆"第n天"和"过n天" | 第n天=过(n-1)天+1 |
| 跨年问题 | 闰年+2的时机 | 必须经过2月才+2 |
| 余数为0 | 误判为第0天 | 余数为0时取周期最后一天 |
| 跨月天数 | 漏算首尾 | 分月计算，注意尾-头+1 |
| 整百年闰年 | 误认1900为闰年 | 整百年必须被400整除 |
