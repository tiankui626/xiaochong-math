### 错题 #1 — 小数除法
- **来源**：课后小测 第2题
- **日期**：2026-06-15
- **错因**：竖式计算错误
- **题目**：(1) $126 \div 45$  (2) $25.46 \div 1.9$
- **正确思路**：
  1. $126 \div 45$：列竖式，$126.0 \div 45 = 2.8$
  2. $25.46 \div 1.9$：除数变整数 → $254.6 \div 19 = 13.4$
- **关键**：小数除法先把除数变整数（同时移动被除数小数点），再仔细列竖式
- **关联知识点**：[[第14讲 小数乘除#3. 小数除法]]

### 错题 #2 — 大量零的小数乘法
- **来源**：课后小测 第3题
- **日期**：2026-06-15
- **错因**：计算小数位数时出错
- **题目**：$\underbrace{0.00\cdots0}_{100个0}128 \times \underbrace{0.00\cdots0}_{120个0}75$
- **正确思路**：
  1. 先算整数部分：$128 \times 75 = 9600$
  2. 数小数位数：第一个数小数点后有 $100+3=103$ 位，第二个有 $120+2=122$ 位，共 $225$ 位
  3. $9600$ 有4位有效数字，但末尾两个0可以抵消2位小数 → 结果为 $0.\underbrace{00\cdots0}_{221个0}96$
- **关键**：小数位数 = 两个因数小数位数之和，再看积的末尾有几个0可以去掉
- **关联知识点**：[[第14讲 小数乘除#2. 小数乘法]]

### 错题 #3 — 幂运算与小数
- **来源**：课后小测 第4题（未做/超时）
- **日期**：2026-06-15
- **错因**：时间不够没做
- **题目**：$(0.1^2 + 0.2^2 + 0.3^2 + 0.4^2)^2 \div (0.1^3 + 0.2^3 + 0.3^3 + 0.4^3)^3$
- **正确思路**：
  1. 先分别算括号内的和：
     - $0.01 + 0.04 + 0.09 + 0.16 = 0.3$
     - $0.001 + 0.008 + 0.027 + 0.064 = 0.1$
  2. 原式 $= 0.3^2 \div 0.1^3 = 0.09 \div 0.001 = 90$
- **关键**：先耐心把每个小数的幂算出来再求和，不要跳步
- **关联知识点**：[[第14讲 小数乘除#3. 小数除法]]

### 错题 #4 — 逆运算求未知数
- **来源**：课后小测 第5题
- **日期**：2026-06-15
- **错因**：多层嵌套逆运算步骤出错
- **题目**：$105.5 + [(40 + \square \div 2.3) \times 0.5 - 1.53] \div (53.6 \div 26.8 \times 0.125) = 187.5$
- **正确思路**：从外往里一层层剥：
  1. 右边 $= 53.6 \div 26.8 \times 0.125 = 2 \times 0.125 = 0.25$
  2. $[\cdots] \div 0.25 = 187.5 - 105.5 = 82$
  3. $(\cdots) \times 0.5 - 1.53 = 82 \times 0.25 = 20.5$
  4. $(40 + \square \div 2.3) \times 0.5 = 22.03$
  5. $40 + \square \div 2.3 = 44.06$
  6. $\square \div 2.3 = 4.06$
  7. $\square = 4.06 \times 2.3 = 9.338$
- **关键**：逆运算从最外层开始，每步只剥一层括号，小心每步的小数计算
- **关联知识点**：[[第14讲 小数乘除#4. 小数四则混合运算]]

### 错题 #5 — 四舍五入与乘积范围
- **来源**：课后小测 第6题（部分对）
- **日期**：2026-06-15
- **错因**：边界范围或计数出错
- **题目**：两个小数四舍五入到个位后分别是7和9，乘积整数部分有多少种取值？
- **正确思路**：
  1. 四舍五入到个位得7 → $6.5 \leq a < 7.5$
  2. 四舍五入到个位得9 → $8.5 \leq b < 9.5$
  3. 最小乘积：$6.5 \times 8.5 = 55.25$
  4. 最大乘积：趋近 $7.5 \times 9.5 = 71.25$（取不到）
  5. 整数部分：$55, 56, \ldots, 71$，共 $71 - 55 + 1 = 17$ 种
- **关键**：四舍五入 → 确定取值范围（注意≤和<的区别），乘积两端确定后数整数个数
- **关联知识点**：[[第14讲 小数乘除#6. 小数点移动的应用题]]
