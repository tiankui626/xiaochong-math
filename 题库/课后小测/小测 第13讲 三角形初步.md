### 第1题

> [!question] 三角形剪角变四边形
> 已知$\triangle ABC$中，$\angle A = 40°$，如图剪一刀，变成四边形，则$\angle 1 + \angle 2 = $ ____°。

- [x] 已掌握

> [!answer]- 点击查看答案
> **答案**：220
> **解析**：
> - $\angle B + \angle C = 180° - 40° = 140°$
> - 剪去后形成四边形，$\angle 1 + \angle 2 = 360° - 140° = 220°$
> **知识点**：[[第13讲 三角形初步#5. 三角形内角和]]、[[第13讲 三角形初步#6. 多边形内角和]]

### 第2题

> [!question] 三边关系求整数三角形种数
> 一个三角形三条边的长度都是整数，其中两条边的长度分别为7厘米与12厘米，共有______种不同的三角形。

- [ ] 未掌握（算错）→ [[错题 第13讲 三角形初步#错题 #1 — 三边关系求种数|查看错题记录]]

> [!answer]- 点击查看答案
> **答案**：13
> **解析**：设第三边为$a$厘米，由三角形三边关系：
> - $a < 7 + 12$，即$a < 19$
> - $a + 7 > 12$，即$a > 5$
> - $a$的取值为$6 \sim 18$之间的整数，共13种
> **知识点**：[[第13讲 三角形初步#3. 三角形三边关系]]

### 第3题

> [!question] 周长为偶数的三角形求第三边
> 一个三角形的周长是偶数，其中的两条边分别是3和2026，则三角形的第三边长为______。

- [ ] 未掌握（少写一种情况，只写了2025）→ [[错题 第13讲 三角形初步#错题 #2 — 周长偶数求第三边|查看错题记录]]

> [!answer]- 点击查看答案
> **答案**：2025或2027
> **解析**：
> - 由三边关系，第三边范围：$2026 - 3 < x < 2026 + 3$，即$2023 < x < 2029$
> - 周长 = $3 + 2026 + x = 2029 + x$为偶数，所以$x$为奇数
> - $x$的取值为：2025、2027
> **知识点**：[[第13讲 三角形初步#3. 三角形三边关系]]

### 第4题

> [!question] 皇冠形角度和
> 如图所示，$\angle 1 + \angle 2 + \angle 3 + \angle 4 + \angle 5 + \angle 6 + \angle 7 + \angle 8 + \angle 9 = $ ____°。

- [x] 已掌握

> [!answer]- 点击查看答案
> **答案**：540
> **解析**：
> - 因为$\angle 7 + \angle 8 = \angle 10 + \angle 11$；$\angle 4 + \angle 5 = \angle 12 + \angle 13$
> - 所以$\angle 1 + \angle 2 + \angle 3 + \angle 4 + \angle 5 + \angle 6 + \angle 7 + \angle 8 + \angle 9$恰好等于五边形的内角和
> - 五边形内角和 = $(5-2) \times 180° = 540°$
> **知识点**：[[第13讲 三角形初步#6. 多边形内角和]]

### 第5题

> [!question] 正五边形与正三角形求角
> 如图，$ABCDE$是正五边形，$CDF$是正三角形，$\angle BFE$等于多少度？

- [x] 已掌握

> [!answer]- 点击查看答案
> **答案**：168°
> **解析**：
> - 正五边形内角均为$108°$，$AB = BC = CD = DE = EA$
> - $CDF$是正三角形，$CD = CF = DF$，$\angle FCD = \angle FDC = 60°$
> - $\angle BCF = 108° - 60° = 48°$
> - 因为$BC = CF$，$\triangle BCF$是等腰三角形，$\angle CBF = \angle DEF = 66°$
> - $\angle ABF = \angle AEF = 108° - 66° = 42°$
> - 四边形$ABFE$内角和为$360°$：
> - $\angle BFE = 360° - 42° - 42° - 108° = 168°$
> **知识点**：[[第13讲 三角形初步#6. 多边形内角和]]

### 第6题

> [!question] 木棒截段不能组成三角形（斐波那契）
> 将一段长50米的木棍，截成很多个小段使得这些小段中任意三段都不能围成三角形，最多能截成多少段？最长的那段至多是多少米？

- [x] 已掌握

> [!answer]- 点击查看答案
> **答案**：最多7段；最长的那段至多30米
> **解析**：
> - 截出的段数尽量多，则每段的长度要尽量小
> - 长度从小到大构成类斐波那契数列：$1, 1, 2, 3, 5, 8, 13, \ldots$
> - 前7项之和：$1 + 1 + 2 + 3 + 5 + 8 + 13 = 33 < 50$
> - 前8项之和：$1 + 1 + 2 + 3 + 5 + 8 + 13 + 21 = 54 > 50$
> - 所以最多7段，最长的那段至多有：$13 + (50 - 33) = 30$（米）
> **知识点**：[[第13讲 三角形初步#4. 斐波那契数列与三边关系]]
