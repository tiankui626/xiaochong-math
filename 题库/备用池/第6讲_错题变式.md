### 变式题 #1-V1
**原题关联：** 错题 #1 — 梯形面积计算
**题目：** 正方形ABCD面积25平方厘米，ED=CH=5厘米，EF=3厘米，EFGH为长方形，求梯形ABGF面积
**答案：** 52.5
**解析：**
1. 正方形边长 $= 5$
2. 上底 $= 5$，下底 $= 5+5+5 = 15$，高 $= 5+3 = 8$
3. 面积 $= (5+15) × 8 ÷ 2 = 80$

让我重新计算：
正方形面积25，边长=5。
ED=CH=5，EF=3。
上底AB=5，下底FG=5+5+5=15？检查一下原题逻辑。

原题：正方形ABCD，ED=CH=4，EF=2。
上底=4，下底=4+4+4=12，高=4+2=6。
面积=(4+12)×6÷2=48。

我的变式：边长=5（面积25），ED=CH=5，EF=3。
上底=5，下底=5+5+5=15，高=5+3=8。
面积=(5+15)×8÷2=20×4=80。

答案：80

### 变式题 #2-V1
**原题关联：** 错题 #2 — 靠墙围长方形最大面积
**题目：** 50米篱笆靠墙，围一条边20米的长方形，最大面积是多少
**答案：** 300 平方米
**解析：**
1. 靠墙一边不用围栏
2. 20米可以是长边（靠墙）或宽边（不靠墙）
3. 若20米靠墙：围栏 = 20 + 2×宽 = 50，宽 = 15，面积 = 300
4. 若20米不靠墙：宽 = 20，长 = 50-40 = 10，面积 = 200
5. 最大面积 = 300
**数学思维：** 分类

### 变式题 #2-V2
**原题关联：** 错题 #2 — 靠墙围长方形最大面积
**题目：** 40米篱笆靠墙围长方形，一条边是12米，求可能的面积
**答案：** 168 或 96 平方米
**解析：**
1. 若12米靠墙：宽 = (40-12)÷2 = 14，面积 = 12×14 = 168
2. 若12米不靠墙：长 = 40-24 = 16，面积 = 12×16 = 192？不对，若12米是宽（不靠墙），则两个宽=24，长=40-24=16，面积=192

但只有两个宽一个长靠墙。若12米是宽，不靠墙，那靠墙的是长=40-24=16，面积=12×16=192。

答案可以是168或192。或题目问"其中一个可能的面积"。

**答案：** 168 或 192
**数学思维：** 分类

### 变式题 #3-V1
**原题关联：** 错题 #3 — 梯形和差问题
**题目：** 梯形面积48平方厘米，高8厘米，下底比上底长4厘米，求上底和下底
**答案：** 上底=5，下底=9
**解析：**
1. 上底 + 下底 = $48 × 2 ÷ 8 = 12$
2. 下底 - 上底 = 4
3. 和差问题：上底 = $(12-4) ÷ 2 = 4$，下底 = 8

等等，$(12-4)/2=4$，下底=4+4=8。对。

答案：上底4，下底8。
**数学思维：** 方程

### 变式题 #3-V2
**原题关联：** 错题 #3 — 梯形和差问题
**题目：** 梯形面积90平方厘米，高6厘米，两底相差8厘米，求两底长度
**答案：** 上底=9，下底=17
**解析：**
1. 上底 + 下底 = $90 × 2 ÷ 6 = 30$
2. 下底 - 上底 = 8
3. 和差问题：上底 = $(30-8) ÷ 2 = 11$，下底 = 19

验证：$(11+19)×6÷2=30×3=90$。对。

答案：上底11，下底19。
**数学思维：** 方程
