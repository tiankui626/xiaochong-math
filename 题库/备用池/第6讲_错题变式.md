# 第6讲 平行四边形与梯形 - 错题变式练习

### 变式题 #1-V1
**原题关联：** 错题 #1 — 梯形面积计算
**题目：** 一个梯形上底7厘米，下底15厘米，高6厘米，求面积。
**答案：** 66平方厘米
**解析：** $(7+15) \times 6 \div 2 = 22 \times 3 = 66$
验证：$66 \times 2 \div 6 = 22 = 7+15$ ✓
**数学思维：** 数形

### 变式题 #1-V2
**原题关联：** 错题 #1 — 梯形面积计算
**题目：** 梯形面积84平方厘米，高7厘米，下底比上底长4厘米，求上底和下底。
**答案：** 上底10厘米，下底14厘米
**解析：**
1. 上底+下底 = $84 \times 2 \div 7 = 24$
2. 下底-上底 = 4
3. 上底 = $(24-4) \div 2 = 10$，下底 = $10+4 = 14$
验证：$(10+14) \times 7 \div 2 = 84$ ✓，$14-10=4$ ✓
**数学思维：** 方程

### 变式题 #2-V1
**原题关联：** 错题 #2 — 靠墙围长方形最大面积
**题目：** 用36米篱笆靠墙围一个长方形花园，面积最大是多少平方米？
**答案：** 162平方米
**解析：**
1. 设宽b，长=36-2b
2. 面积 = $(36-2b) \times b$，令c=2b, d=36-2b，c+d=36
3. cd最大=18×18=324，面积=324÷2=162
验证：b=9, 长=18, 面积=18×9=162 ✓
**数学思维：** 化归

### 变式题 #2-V2
**原题关联：** 错题 #2 — 靠墙围长方形最大面积
**题目：** 用44米篱笆靠墙围长方形，其中一条边是12米，两种情况下面积分别是多少？
**答案：** 192平方米或240平方米
**解析：**
1. 若12米靠墙（长=12）：宽 = $(44-12) \div 2 = 16$，面积 = $12 \times 16 = 192$
2. 若12米不靠墙（宽=12）：长 = $44 - 12 \times 2 = 20$，面积 = $12 \times 20 = 240$
验证：192→围栏=12+16×2=44 ✓；240→围栏=20+12×2=44 ✓
**数学思维：** 分类
