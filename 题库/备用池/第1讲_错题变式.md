# 第1讲 方田探秘 - 错题变式练习

### 变式题 #1-V1
**原题关联：** 错题 #1 — 田字格蝴蝶模型
**题目：** 一个大长方形被一条横线和一条竖线分成四个小长方形。已知左上面积15，右上面积45，左下面积20，求大长方形的面积。
![[第1讲_V2_田字格进阶.png]]
**答案：** 140
**解析：**
1. 田字格对角乘积相等：$15 \times 右下 = 45 \times 20 = 900$
2. 右下 $= 900 \div 15 = 60$
3. 大长方形面积 $= 15 + 45 + 20 + 60 = 140$
验证：$15 \times 60 = 900 = 45 \times 20$ ✓
**数学思维：** 转化

### 变式题 #2-V1
**原题关联：** 错题 #3 — 构造弦图
**题目：** 长方形面积96平方厘米，内部四边形MNPQ通过构造辅助线，中间形成的小矩形面积为6平方厘米。求四边形MNPQ面积。
![[第1讲_V3_弦图.png]]
**答案：** 51
**解析：**
1. 设四角三角形面积和为S，四边形面积为x
2. $x + S = 96$，$x - S = 6$
3. $x = (96 + 6) \div 2 = 51$
验证：$S = 96 - 51 = 45$，$51 - 45 = 6$ ✓
**数学思维：** 整体
