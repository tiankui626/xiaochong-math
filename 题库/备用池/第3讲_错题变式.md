# 第3讲 何时无余 - 错题变式练习

### 变式题 #1-V1
**原题关联：** 错题 #1 — 33的整除特征
**题目：** 六位数 $\overline{5247a3}$ 能被33整除，求a的值。
**答案：** 3
**解析：**
1. 33的整除特征：两位一分段求和
2. 52 + 47 + $\overline{a3}$ = 52 + 47 + 10a + 3 = 102 + 10a
3. 需要被33整除：102 + 10a = 132（33×4）
4. 10a = 30，a = 3
验证：524733 ÷ 33 = 15901 ✓
**数学思维：** 转化

### 变式题 #2-V1
**原题关联：** 错题 #2 — 数字和凑9的倍数
**题目：** 在314右边连续添加数码5，第一次得到9的倍数时，一共添加了多少个5？
**答案：** 2
**解析：**
1. 原数字和：3 + 1 + 4 = 8
2. 每添一个5，数字和增加5
3. 需要 8 + 5k 是9的倍数：k=1→13，k=2→18 ✓
4. 添加2个5，得到31455
验证：31455的数字和 = 3+1+4+5+5 = 18，18÷9=2 ✓
**数学思维：** 同余

### 变式题 #3-V1
**原题关联：** 错题 #3 — 拼接数的整除
**题目：** 将自然数1,2,3…依次写下去组成多位数123456789101112…，写到哪个自然数时第一次能被72整除？（72=8×9）
**答案：** 36
**解析：**
1. 被9整除：$n(n+1)/2$ 是9的倍数 → n或n+1是9的倍数
2. 被8整除：末三位是8的倍数
3. 候选：n=8,9,17,18,26,27,35,36…
4. n=36时末三位为536，536÷8=67 ✓
5. $36×37/2=666$，数字和6+6+6=18，是9的倍数 ✓
验证：72=8×9，536÷8=67✓，666÷9=74✓
**数学思维：** 分解转化
