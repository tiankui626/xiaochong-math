# 第4讲 深思熟虑 - 错题变式练习

### 变式题 #1-V1
**原题关联：** 错题 #1 — 删数求最小
**题目：** 从 9876543210 中划去5个数字（不改变顺序），得到最小的5位数是多少？
**答案：** 43210
**解析：**
1. 保留5个数字，首位不能为0
2. 数字序列：9,8,7,6,5,4,3,2,1,0
3. 首位尽量小：取4（第6位），后面还有3,2,1,0共4个数字，刚好凑5位
4. 取3作首位？第7位是3，后面只有2,1,0共3个，不够4位
5. 所以首位最小取4，答案43210
验证：从9876543210中保留4,3,2,1,0，顺序不变 ✓
**数学思维：** 贪心

### 变式题 #2-V1
**原题关联：** 错题 #2 — 积一定差小和小
**题目：** $(a+3) \times (b+3) = 180$，a、b是不同的自然数，求 $a+b$ 的最小值。
**答案：** 21
**解析：**
1. 180的因数对（两数最接近）：12×15=180
2. a+3=12, b+3=15 → a=9, b=12
3. a+b = 9+12 = 21
验证：(9+3)×(12+3) = 12×15 = 180 ✓，a≠b ✓
**数学思维：** 转化

### 变式题 #3-V1
**原题关联：** 错题 #4 — 拆分问题
**题目：** 把76拆成5个互不相同的正整数之和，最大的数最大是多少？最大的数最小是多少？
**答案：** 最大的最大=66，最大的最小=18
**解析：**
1. 最大的最大：其余4个取最小 1+2+3+4=10，最大=76-10=66
2. 最大的最小：尽量平均，76÷5=15.2，取 13+14+15+16+18=76，最大=18
验证：1+2+3+4+66=76 ✓；13+14+15+16+18=76 ✓，5个互不相同 ✓
**数学思维：** 分类

### 变式题 #4-V1
**原题关联：** 错题 #5 — 靠墙围面积
**题目：** 用48米围栏靠墙围一个长方形花园，面积最大是多少平方米？
**答案：** 288
**解析：**
1. 设宽b，长=48-2b（靠墙的是长边）
2. 面积 = (48-2b)×b
3. 令c=2b, d=48-2b，则c+d=48
4. 和一定差小积大：c=d=24时cd最大=576
5. 2b×(48-2b)=576，面积=b×(48-2b)=576/2=288
验证：b=12, 长=24, 面积=24×12=288 ✓
**数学思维：** 化归
