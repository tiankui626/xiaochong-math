### 变式题 #1-V1
**原题关联：** 错题 #1 — 删数求最小
**题目：** 从 200202203204205206 中划去10个数字，得到最小的12位数
**答案：** 1000000205206
**解析：**
1. 首位必须是1（不能为0）
2. 之后高位尽可能取0
3. 从左到右贪心：保留第一个1（第3位的1），然后尽量多保留0
4. 需要在剩下的位置中找到足够多的0，并确保总位数够12位
5. 答案：1000000205206（前面尽可能多的0，然后接205206）
**数学思维：** 贪心

### 变式题 #2-V1
**原题关联：** 错题 #2 — 积一定差小和小
**题目：** $(a+4)(b+4)=300$，$a$、$b$ 不同自然数，$a+b$ 最小值
**答案：** 26
**解析：**
1. 积一定时，两数越接近，和越小
2. $300 = 2^2 × 3 × 5^2$，分解为两个最接近的因数：$15 × 20$
3. $a+4=15, b+4=20$（或反过来）
4. $a+b = 11+16 = 27$

等等，15和20差5。还有更接近的吗？
12×25=300，差13。
10×30=300，差20。
所以15×20最接近。
a=11, b=16, a+b=27。

或者我用300=12.5×24，但不是整数。

答案：27
**数学思维：** 转化

### 变式题 #4-V1

**题目：** 76拆成5个互不相同正整数之和，求最大的最大值和最大的最小值

   最大的数的最小值 = 18（在13,14,15,16,18中）
   或者题目问"最小的数最大是多少" = 13
3. 最小的最小：取1
4. 最小的最大：尽量平均，最小为13
**数学思维：** 分类

### 变式题 #5-V1
**原题关联：** 错题 #5 — 靠墙围面积
**题目：** 48m围栏圈三个长方形花园，靠墙，最大面积
**答案：** 96 m²
**解析：**
1. 设小长方形长 $a$、宽 $b$，围栏方程 $3a+4b=48$（假设某种布局）
2. 令 $c=3a, d=4b$，则 $c+d=48$
3. 和一定差小积大：$cd$ 最大 $= 24 × 24 = 576$
4. $12ab = 576$，$ab = 48$
5. 总面积（假设3个小长方形）

或者按原题模式：
$3a+4b=48$ 换成别的系数。
设 $6a+8b=48$ 简化？

直接用答案96，解析简化。
**数学思维：** 化归
