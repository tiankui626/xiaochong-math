# 第10讲 简易方程（未知之数）

## 核心知识点

### 1. 认识方程

> [!tip] 基本概念
> - **等式**：用等号连接的表示相等关系的式子
> - **方程**：含有未知数的等式
> - **方程的解**：使等式成立的未知数的值

判断是否为方程的两个条件：
1. 含有未知数
2. 是等式（有等号）

> [!warning] 易错判断
> - $1+3=4$：❌ 没有未知数
> - $2x+5$：❌ 不是等式（没有等号）
> - $x+3>4$：❌ 不是等式（是不等式）
> - $x+y=10$：✅ 含未知数的等式
> - $x=2$：✅ 也是方程

### 2. 等式性质

> [!tip] 等式性质1
> 等式两边同加同减，左右两边仍然相等。
> $$若 a = b，则 a \pm c = b \pm c$$

> [!tip] 等式性质2
> 等式两边同乘同除以同一个数（非零数），左右两边仍然相等。
> $$若 a = b，则 a \times c = b \times c \quad (c \neq 0)$$
> $$若 a = b，则 a \div c = b \div c \quad (c \neq 0)$$

### 3. 解方程书写规范

> [!important] 书写规范四条
> 1. 先写"解"
> 2. 等号对齐
> 3. 最后答案 $x$ 写在等号左边
> 4. 检验

## 模块一：移项

### 过"桥"变号

> [!tip] 移项规则
> 把方程中的某一项从等号一边移到另一边时，要**变号**：
> - 加号变减号，减号变加号
> - 乘号变除号，除号变乘号

**简单方程示例**：
- $x + 7 = 16 \Rightarrow x = 16 - 7 = 9$
- $x - 11 = 3 \Rightarrow x = 3 + 11 = 14$
- $5x = 45 \Rightarrow x = 45 \div 5 = 9$
- $x \div 7 = 8 \Rightarrow x = 8 \times 7 = 56$

### 两步方程（含系数和常数）

步骤：先移常数项，再处理系数。

- $2x + 6 = 10$：$2x = 10 - 6 = 4$，$x = 4 \div 2 = 2$
- $4x - 5 = 19$：$4x = 19 + 5 = 24$，$x = 24 \div 4 = 6$

### 未知数在减号后面（小注大集）

当 $x$ 在减号后面时，需要注意方向：
- $20 - x = 16$：$20 - 16 = x$，$x = 4$

> [!warning] 口诀
> "小往大靠，'-'往'+'靠"——含 $x$ 的项移到系数大的一边，减号的项移到加号一边。

### 合并同类项后移项

先把等号同侧的同类项合并，再移项。
- $3x + 2x = 15$：$5x = 15$，$x = 3$
- $5x - 2x + 6 = 9$：$3x = 9 - 6 = 3$，$x = 1$

### 两边都有未知数的方程

将未知数移到同一边，常数移到另一边：
- $4x + 3 = 3x + 8$：$4x - 3x = 8 - 3$，$x = 5$
- $5x + 4x + 3 = 3x + 2x + 27$：$9x + 3 = 5x + 27$，$4x = 24$，$x = 6$

## 模块二：去括号

### 去括号规则

> [!tip] 同级运算
> - **加减同级**：括号外是减号或除号时，去括号里面要变号
>   - $a + (a+2) = a + a + 2 = 2a + 2$
>   - $a - (a-2) = a - a + 2 = 2$
> - **乘除同级**：直接结合
>   - $6 \times (m \times 5) = 6 \times m \times 5 = 30m$
>   - $12 \div (m \div 6) = 12 \div m \times 6$

> [!tip] 不同级运算（乘法分配律）
> $$a \times (b + c) = ab + ac$$
> $$a \times (b - c) = ab - ac$$

> [!warning] 口诀
> "是减号（除号），吓一跳，括号里面要变号"

### 含括号方程的解法

步骤：**先去括号** → **再移项** → **合并** → **求解**

- $3(x+3) = 15$：$3x + 9 = 15$，$3x = 6$，$x = 2$
- $4(x-6) = 12$：$4x - 24 = 12$，$4x = 36$，$x = 9$
- $12 - (3x-4) = 4$：$12 - 3x + 4 = 4$，$16 - 4 = 3x$，$x = 4$
- $12x - 2(5+x) = 20$：$12x - 10 - 2x = 20$，$10x = 30$，$x = 3$

### 多层括号

- $2 + 3(x-26) = 52 - (x-40)$
- $15 - 2(8-5x) = 3(10+x) - 2(2-2x)$

## 模块三：含除法的方程（通分/去分母）

> [!tip] 方法
> 等式两边同乘以分母的最小公倍数，将除法转化为乘法。

**示例**：
- $x \div 2 + x = 30$：两边同乘2，得 $x + 2x = 60$，$3x = 60$，$x = 20$
- $x \div 2 + x \div 3 = 30$：两边同乘6，得 $3x + 2x = 180$，$5x = 180$，$x = 36$
- $3(x+6) \div 4 = (4x-5) \div 6 + (2x+2) \div 3$：同乘12

## 模块四：方程组

### 代入消元法

> [!tip] 方法
> 将一个方程表示成 $x = \ldots$ 或 $y = \ldots$ 的形式，代入另一个方程。

**示例**：
$$\begin{cases} x = y + 2 \\ x + 3y = 14 \end{cases}$$
将①代入②：$(y+2) + 3y = 14$，$4y = 12$，$y = 3$，$x = 5$

### 加减消元法

> [!tip] 方法
> 找相同系数的项，"**同号用减，异号用加**"消去一个未知数。

**示例**：
$$\begin{cases} 3x + 4y = 7 \\ 7x + 4y = 11 \end{cases}$$
②$-$①得：$4x = 4$，$x = 1$，代回得 $y = 1$

> [!tip] 特殊技巧：加减后整体除
> $$\begin{cases} 17x + 19y = 91 \\ 19x + 17y = 89 \end{cases}$$
> ①$+$②得 $36x + 36y = 180$，即 $x + y = 5$
> ②$\times 1$再减①得 $2x - 2y = -2$，即 $x - y = -1$
> 解得 $x = 2, y = 3$

## 拓展：新定义运算与方程

利用新定义的运算规则列方程求解：
- 规定 $\langle a, b, c, d \rangle = 2ab - c + d$，则 $\langle 1, 3, 5, x \rangle = 7$ 即 $2 \times 1 \times 3 - 5 + x = 7$，$x = 6$
- $a@b = 3a - 2b$，当 $x@5$ 比 $5@x$ 大5时，$3x - 10 - (15 - 2x) = 5$，$x = 6$

## 解题策略

1. **先观察方程结构**：确定是哪种类型（移项、去括号、去分母、方程组）
2. **去括号**（如有）：注意符号变化
3. **移项**：未知数归一边，常数归另一边
4. **合并同类项**
5. **系数化为1**：两边同除以系数
6. **检验**：将答案代回原方程验证

## 易错点

> [!warning] 注意
> 1. **去括号变号**：括号前是减号，括号内每一项都要变号
> 2. **移项变号**：从等号一边移到另一边要变号
> 3. **未知数在右边**：最后要写成 $x = \ldots$ 的形式
> 4. **去分母要每项都乘**：不能漏乘某一项
> 5. **方程组代入后要回代**：求出一个未知数后要代回求另一个
> 6. **含除法要找公倍数**：多个分母时取最小公倍数
